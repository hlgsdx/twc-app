# TWC App

TWC App is a Flutter client for exploring the Tsukuba Web Corpus (TWC) interface. It provides a mobile and desktop-friendly way to search headwords, inspect headword details, and drill down into collocation and example sentence results.

## Status

This project is still a work in progress (WIP). Some areas of the UI and feature set are incomplete or under active development.

## What the App Does

- Search TWC headwords from the home screen
- Browse the full headword list when no query is entered
- View headword detail pages with frequency and summary information
- Explore collocation groupings and pattern frequency views
- Open example sentence lists for a selected collocation
- Keep simple search history for quicker repeat queries
- Provide placeholder navigation for Favorites and Settings while those areas continue to evolve
